Roman Numerals API

REST API to convert numbers following Roman numerals. Allows conversion from Roman to decimal, or from decimal to Roman. No API key needed.


Usage


Convert from decimal number to roman number (from 1 to 4000)

        
        GET https://{serverurl}/decimalToRoman?number={numberToConvert} HTTP/1.1

Convert from roman number to decimal number (letters must be capital)

        
        GET https://{serverurl}/romanToDecimal?number={numberToConvert} HTTP/1.1
